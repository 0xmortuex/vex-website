Everything you actually use

One browser, fully yours

No accounts, no telemetry, no bloat — just the features that make browsing faster and calmer.

Vertical tabs & groups

A clean sidebar with drag-to-reorder tabs and collapsible, color-coded groups that keep dozens of tabs sane.

Built-in AI agent

Summarize pages, ask questions about what you're reading, translate, and let it click and type to finish tasks for you.

Ad & tracker blocking

Pattern-based blocking is on by default, so pages load faster and trackers don't follow you around the web.

Command bar

Hit Ctrl+K to search, navigate, switch panels, or run any action — without lifting your hands off the keyboard.

Workspaces & sessions

Switch between Work, School, Dev and Personal contexts, and save or restore named sets of tabs whenever you need them.

Tab sleep & memory

Inactive tabs auto-sleep to near-zero RAM, with a memory panel to wake, sleep, or close them on demand.

Split screen & PiP

View two tabs side by side, or pop any video into a floating always-on-top window while you keep working.

Focus, gestures & more

Focus sessions that block distractions, mouse gestures, bookmarks, an RSS reader, read-aloud, container tabs, and themes galore.

Sync, handoff & passwords

End-to-end encrypted sync, send tabs between desktop and Vex Mobile, and a password manager locked with your OS keychain.

Peek, Boosts & Skills

Shift+click links to preview them in a floating overlay, zap annoying elements off any site forever, and run saved AI commands on any page.

New in 2.27

Now it does even more

Copy from anywhere, even sites that block it — plus a customizable sidebar and shortcuts, theme-matched tab groups, reading tools, privacy hardening, on-device AI, and developer utilities, all built in.

Copy from anywhere

Sites that disable text selection, copying, and right-click? Unlock any page in one tap from the new quick-tools menu in the toolbar, or flip on the global switch to bypass those blocks everywhere. Your browser, your text.

Make it yours

Right-click any left-sidebar button to rename it, change its icon, relink it, hide it, or reorder the whole bar — or manage them all in Settings. Pin any site as a panel, customize your home-page shortcuts, and pick tab-group colors that match every theme.

Read smarter

Highlights that persist across visits, full-text Recall to find any page by what it said, a one-word-at-a-time speed reader, bionic reading, and dyslexia-friendly fonts.

Privacy hardening

Fingerprint randomization, DNS-over-HTTPS (Cloudflare/Google/Quad9), and a Privacy Report that shows exactly which trackers follow you across sites.

On-device AI

Run a small language model entirely on your machine via WebGPU — private, offline, no server. Or keep using cloud and local Ollama; Vex routes intelligently.

AI memory & MCP

Tell the assistant facts to remember in every chat, and connect Model Context Protocol servers so the agent can call external tools to get things done.

Screenshot → Code

Capture any page and have AI rebuild it as a clean HTML, Tailwind, or React file — preview it live in a new tab or copy the code.

Dev & power tools

A built-in API client with a JSON tree viewer, multi-viewport responsive preview, page-change monitoring, and one-click Wayback Machine archiving.

Logins that just work

Sign in with Discord, Google, Microsoft and other "Login with…" buttons now complete reliably — in a tidy in-app window that stays connected to your tab, so no more blank popups or "something went wrong" dead-ends.

Read & grab anything

Select text for a one-tap Explain / Summarize / Translate bar. Get past paywalls with Read Free — reset metered walls or open a free archived copy. And save the video or audio playing on a page. Plus a much faster address bar.

Search, watch, control

Type keyword searches & bangs right in the address bar (yt, gh, !w…). A Master Volume slider (up to 500%) for every tab. Netflix, Prime Video & Disney+ in the sidebar. And right-click any image to zoom & pan it.

Discord — even when it's blocked

A Discord panel in the sidebar with a one-click block bypass for regions where it's censored. Hit Auto-configure and Vex finds the desync mode that gets through your network — no extra app, no admin. Plus one-click Vencord for plugins & themes.
